Important FAQ

What is the Document Object Model(DOM)?

Document Object Model(DOM) narrate HTML document in a hierarchical tree sturcture manner, where each and every attribute, element and text component is considered as a node.

How do you select an element from the DOM?

There are several ways to get an element. Such as:

getElementById
'const element = document.getElementById('element');'

getElementsByTagName
const elements = document.getElementsByTagName('div');

querySelector
'const element = document.querySelector('.className');

What is event delegation in the context of the Dom, and why is it useful?

In JavaScript, event listener attached to parent element and it manage its multiple child element. It help to improve performance, lessen complexity and handle dynamic content automatically.

How do you manipulated an element's attributes and styles using the DOM?

Element's attributes can be manuplated by using 'getAttribute', 'setAttribute', 'removeAttribute' and can be styled by using 'style' which are the methods of DOM.
